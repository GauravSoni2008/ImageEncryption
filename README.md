# 🖼️ Image Encryption

A Python-based application that securely encrypts and decrypts images using cryptographic techniques. This project provides a simple graphical interface, allowing users to protect sensitive images from unauthorized access while ensuring they can be restored without any loss using the correct encryption key.

## ✨ Features

* 🔐 Encrypt image files securely
* 🔓 Decrypt encrypted images using the correct key
* 🖼️ Supports common image formats
* 🖥️ Simple and intuitive GUI
* ⚡ Fast image processing
* 💾 Save encrypted and decrypted images
* 🔑 Password/Key-based encryption

## 💡 Usage

1. Launch the application.
2. Select the image you want to encrypt.
3. Enter an encryption key/password.
4. Click **Encrypt** to generate the encrypted image.
5. To restore the original image, select the encrypted image, enter the same key, and click **Decrypt**.

---

## 🔒 How It Works

The application transforms the image data using an encryption algorithm and a user-provided key. The encrypted image appears unreadable until the correct key is used for decryption, ensuring secure storage and transfer of sensitive images.
